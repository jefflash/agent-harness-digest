## Top Stories

**1. Claude Fable 5 and Claude Mythos 5 — Anthropic (Imp 5)**

Anthropic launched Claude Fable 5 and Claude Mythos 5 today, marking the first public release of a Mythos-class model. Fable 5 is state-of-the-art on nearly every tested benchmark — software engineering, knowledge work, vision, scientific research, and long-horizon agentic tasks — but ships with safety classifiers that automatically fall back to Claude Opus 4.8 for queries touching cybersecurity, biology/chemistry, or suspected distillation (affecting fewer than 5% of sessions). Claude Mythos 5, the same underlying model with cyber safeguards lifted, remains restricted to Project Glasswing cybersecurity partners and, soon, select biology researchers. Both models are priced at $10/M input and $50/M output tokens — less than half the price of Mythos Preview — and are available now on the Claude API and consumption-based Enterprise plans; subscription plans get free access through June 22 before usage credits kick in. In early tests, Stripe reported Fable 5 compressed months of engineering into days on a 50M-line Ruby codebase migration, while Hebbia called it the highest-scoring model on their senior-analyst finance benchmark.

**2. OpenClaw 2026.6.5 Stable — OpenClaw (Imp 3)**

OpenClaw shipped its first stable release on the June YYYY.M.PATCH train, 2026.6.5, promoting the June beta series to a stable build. Key improvements include channel-safety hardening (QQBot now strips model `<thinking>` scaffolding before delivery), MCP tool result coercion that prevents Anthropic 400 errors and poisoned session history when tools return richer content types, recovery of extended-thinking sessions after prompt-cache expiry or Gateway restart, and new Parallel web search support. Auth and plugin install durability also improved, alongside CI and packaging tightening. This release supersedes the 2026.6.2-beta.1 previously tracked.
