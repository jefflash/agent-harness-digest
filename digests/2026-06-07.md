## Top Stories

**1. Hermes Agent v0.16.0 "The Surface Release" — NousResearch (Imp 4)**
NousResearch shipped v0.16.0 on June 5 with 874 commits, 542 merged PRs, and 399 closed issues — its largest release yet. The marquee addition is Hermes Desktop, a native Electron app for macOS, Linux, and Windows that installs in one click, updates itself in-app, supports drag-and-drop files, concurrent multi-profile sessions, and connects to a remote Hermes gateway over OAuth. A full browser-based admin panel (MCP catalog, messaging channels, credentials, webhooks, OIDC login), first-time "Quick Setup via Nous Portal" onboarding, and a CVE-2026-48710 security fix round out the release.
